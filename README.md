# Tailor Management System

This is a web-based application developed using Python and Flask to manage tailoring shop operations efficiently.

## Features
- Add customer details
- Edit customer information
- Search customers
- Manage measurements
- Simple and user-friendly interface

## Technologies Used
- Python
- Flask
- SQLite (tailor.db)
- HTML (templates)

## How to Run
1. Install Flask
2. Run app.py
3. Open browser and go to localhost
