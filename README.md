# Tailor Management System

Tailor Management System is a web‑based application built with Python and Flask to help tailoring shops manage customer information and measurements efficiently.

## 📌 Features
- Add and store customer details
- Edit existing customer information
- Search for customers
- Manage measurements
- Simple and easy‑to‑use interface

## 🧰 Technologies Used
- Python  
- Flask  
- SQLite (tailor.db)  
- HTML (templates)

## 🚀 How to Run
1. Clone the repository  
   `git clone https://github.com/hemanthrajkumarar2007/tailor-management-system.git`
2. Navigate to the folder  
   `cd tailor-management-system`
3. Install Flask  
   `pip install flask`
4. Run the application  
   `python app.py`
5. Open your browser and go to `http://localhost:5000`
