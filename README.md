# tailor-management-system
A simple Tailoring Shop Management System built using Python and Flask.  This application allows tailors to store customer details, shirt and pant measurements, search customers by bill number or phone number, edit measurements, and manage customer records easily.  Designed to run on mobile devices or tablets for shop use.


 # Tailor Shop Management System

## Overview
This project is a simple Tailoring Shop Management System built using Python and Flask. 
It helps tailors store and manage customer measurements digitally instead of writing in notebooks.

## Features
- Add customer details
- Store shirt and pant measurements
- Search customers by bill number or phone number
- Edit customer measurements
- Delete customer records

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- VS Code

## How to Run
Install Flask:
pip install flask

Run the program:
python app.py

Open in browser:
http://127.0.0.1:5001
